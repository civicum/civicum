# Ilusión Laboral / Carga Falsa (Squad 276)

**Misión:** ¿Usamos animaciones de carga para que la IA simule "Pensar"?

## 1. El Riesgo del Teatro Mecánico
CIVICUM es rápido (Next.js Edge). Si el ciudadano de Quellón somete su queja a la Civia-AI para resumirla, y la IA la devuelve instantáneamente en (100 milisegundos), la psicología del UX nos dice que el usuario sentirá que *"Lo despacharon muy fácil"* y no lo leyeron con *"suficiente respeto legal"*.
Esto ha llevado a bancos y aerolíneas a meter `SetTimeout(2000)` para dibujar un *Spinner Falso* diciendo "Buscando al mejor asesor...". Esto es *Theater Security* o *Labor Illusion*. Pero éticamente es una mentira al ciudadano. Mentir contraviene todo CIVICUM.

## 2. El Esfuerzo Transparente (Veracidad Procesal)

1.  **Prohibición de Pseudo-Retrasos:** El Front-End no introducirá falsedades intencionadas para engañar al cerebro límbico humano.
2.  **Streaming Verdadero (LLM Token by Token):** En lugar de un Spinner de humo, el usuario recibe el procesamiento en Crudo. El componente Server Sent Events asíncrono imprime las palabras orgánicas *a la misma velocidad de procesamiento del LLM Tensor (Word-by-word streaming)*. El retraso que ve el contribuyente, es el retraso biológico real del silicio razonando sus datos. Honestidad brutal.
3.  **Prohibición de "Sello Mágico":** Civia jamás emitirá frases del calibre: *"Nuestros expertos legales virtuales han analizado el caso."* Solo publicará: *"Sistema Automatizado Civia Model gpt-4o Min ha extraído 4 conclusiones semánticas."*.
