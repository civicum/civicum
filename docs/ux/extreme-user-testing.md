# Protocolo de Prueba de Usuario Final (Squad Omega-Final)

**Squad:** 100. Escuadrón Omega-Final: La Verdad del Usuario  
**Misión:** Test real con 5 usuarios extremos (S1 a S6). La prueba final.  
**Responsables:** UX Researcher, Service Designer, Riesgos, Product Lead.

## 1. Los 5 Jinetes del Apocalypse UX
Reclutamos a 5 usuarios para una prueba de fuego de 1 semana:
1.  **El Escéptico:** Odia la política y cree que todos roban.
2.  **El Abuelo:** No ve bien y le tiemblan las manos.
3.  **El Activista:** Quiere usar la plataforma para su agenda política radical.
4.  **El Hacker:** Intentará romper la seguridad por diversión.
5.  **El Vecino Invisible:** Vive en una toma, sin dirección formal ni correo.

## 2. Criterio de Éxito: "Nadie se rompe"
- La app debe servirles a todos sin que ninguno rompa el sistema ni se frustre hasta el abandono.
- Si el "Abuelo" no puede reportar, fallamos.
- Si el "Hacker" inyecta SQL, fallamos.

## 3. La Pregunta Final (NPS Ético)
No preguntamos "¿Recomendarías la app?".
Preguntamos: "¿Crees que tu barrio es un lugar mejor gracias a esta herramienta?".

*Si la respuesta es Sí, estamos listos.*
