# IPFS Backup Incensurable (Squad 324)

**Misión:** Guardar evidencia forense en IPFS (InterPlanetary File System) para que la PDI o Estado nunca rutee el borrado mediante orden judicial estadounidense (Takedown a S3 AWS).

## 1. El Riesgo del Takedown Constitucional Subrogado (The Big Brother Hack)
Una comuna desclasifica un contrato que comprueba actos flagrantes de colusión entre la Policía y una constructora para tapar crímenes forestales. Guardan el documento probatorio PDF en CIVICUM (AWS S3 y Vercel).
El Ministerio del Interior de Chile, coludido, apela mediante abogados lobbistas a un Tribunal de Seattle argumentando "Violencia Institucional Privada Mentirosa" contra Vercel Inc. / Amazon. Amazon (O Cloudflare), por sus Terms Of Service, bloquea, cierra y tumba el Bucket de CIVICUM en un `Takedown de 24 horas`. Borraron toda la historia de Chile apretando un botón en California. 

## 2. Inmutabilidad Descentralizada Espacial
Si el estado nos rompe S3 Cloud, nosotros ya no estamos ahí. 

1.  **Inyección Doble (Web2 / Web3 Bridge):** Cuando una Denuncia es catalogada como `[Urgency Level: MAX / Corrupción]`. El Frontend no solo empuja a Vercel Blob. Por debajo, ejecuta una biblioteca `.js` (`Helia` o IPFS Desktop Bridge) asíncrona subiendo los bytes fotográficos al InterPlanetary File System, convirtiendo ese PDF de Corrupción Policial en un identificador Hash `CID (Content Identifier)`. 
2.  **Riego Esporal (Pinning Service Descentralizado):** CIVICUM paga micropagos Crypto (`Filecoin` / `Arweave`) o un Pinning Cluster en Suiza (`Proton`/`Pinata`) usando la API key comunal. Ahora, el PDF corrupto policial existe copiado y trozado en el disco duro de miles de estudiantes suizos, rumanos y asiáticos de la red P2P global. 
3.  **URL Redireccionador Burlón (The Hydra Link):** Si Amazon S3 responde Código 404 (Obedecieron la Censura del Gobierno). CIVICUM de forma transparente lanza y envuelve al ciudadano a la puerta de enlace global pública: `https://ipfs.io/ipfs/{CID}`. Imposible de borrar desde Chile, imposible de enjuiciar desde California. La evidencia ya no nos pertenece, le pertenece a la humanidad y el gobierno debe rendirse a la verdad fáctica incontrolable.
