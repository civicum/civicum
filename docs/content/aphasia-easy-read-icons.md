# Protocolo Afasia y Lectura Fácil (Squad Bio-4)

**Squad:** 159. Escuadrón Afasia: Lectura Fácil  
**Misión:** ¿Puede entender la app alguien que perdió capacidad lingüística post-ACV?  
**Responsables:** Editor Cívico, UI Visual, Accesibilidad.

## 1. "Icon-First" Design
- Ninguna acción crítica depende exclusivamente de texto.
- **Bache =** Icono de calle rota + Texto "Bache".
- **Enviar =** Icono de avión de papel/flecha + Texto "Enviar".
- Si le quitas el texto a la app, debe seguir siendo navegable al 80%.

## 2. Oraciones Claras (Sujeto + Verbo + Predicado)
- Prohibida la voz pasiva ("El reporte ha sido enviado").
- Permitido: "Tú enviaste el reporte" o "Enviado con éxito".
- Nivel de lectura: 6to básico. Sin palabras esdrújulas innecesarias.

## 3. Botones de "Estoy Confundido"
- En flujos largos, siempre hay un botón de escape visible ("Volver al inicio" o "¿Ayuda?") que no requiere leer instrucciones complejas para salir de un callejón sin salida.

*La dignidad es entender lo que firmas y lo que reportas.*
