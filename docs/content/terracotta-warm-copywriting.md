# Copywriting con Alma / Tono Terracota (Squad 232)

**Misión:** Revisar que ningún mensaje suene a robot o burócrata. Calidez "Terracota".

## 1. El Riesgo de la Frialdad Institucional
El ciudadano ya teme al Estado y a la burocracia policial. Si la interfaz web de CIVICUM usa el lenguaje `Error Code 401: Unauthorized Handler Input`, u `Operación Fallida. Reintente`, se confirma el sesgo subconsciente de que la plataforma es una barrera inhumana más.

## 2. El Manual de Tono de Voz "Terracota"
El color Terracota del diseño (Cálido chileno, Tierra) debe permear la sintaxis de las alertas, botones (Call to Action) y *Toast Notifications*.

### Reemplazos Estrictos (Anti-Burocracia):
*   ❌ **Estado Actual:** `No hay proyectos encontrados en su área geográfica de búsqueda.`
*   ✅ **Tono Terracota:** `En este momento no hay iniciativas aquí. ¡Sé la primera persona en alzar la voz por tu cuadra!`
*   <hr>
*   ❌ **Notificación Error DB:** `Ups! Request timeout exception. Error interno.`
*   ✅ **Tono Terracota:** `Chuta, los cables andan lentos hoy. Tratamos de guardar tu aporte pero no alcanzó a llegar. Por favor, aprieta 'enviar' de nuevo.`
*   <hr>
*   ❌ **Formulario Obligatorio:** `Campo de RUT Inválido/Requerido para comprobación de identidad nacional.`
*   ✅ **Tono Terracota:** `Necesitamos tu RUT solo para asegurar que un robot no esté votando por ti. (Nadie en la plaza sabrá tu número).`
