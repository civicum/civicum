# Protocolo de Desastres Naturales (Squad Medivac-4)

**Squad:** 89. Escuadrón Medivac-4: Desastres Naturales  
**Misión:** ¿Sirve la app durante un terremoto? (Info oficial vs caos).  
**Responsables:** GIS, Comms, Low-end, Fuentes Públicas.

## 1. Modo Terremoto (UI Lite)
Si la ONEMI (Senapred) declara alerta roja:
- La app activa "Modo Desastre":
    - Mapa vectorizado simple (sin texturas pesadas).
    - Carga solo texto (sin fotos/videos de gatos).
    - Objetivo: Funcionar con redes 2G saturadas y ahorrar batería del usuario.

## 2. Fuente Única de Verdad
- En el header aparece un banner inamovible con la última instrucción oficial de Senapred/SHOA ("Evacuar borde costero").
- Los reportes ciudadanos se verifican contra esta verdad oficial antes de ser viralizados.

## 3. "Estoy Bien" (Safety Check)
- Botón gigante de un toque para marcarse como "A salvo".
- Genera un SMS comprimido a la familia si no hay datos.

*En un terremoto, los megabytes son tan valiosos como el agua.*
