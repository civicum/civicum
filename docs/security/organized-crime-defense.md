# Protocolo de Defensa contra Crimen Organizado (Squad Zero-2)

**Squad:** 142. Escuadrón Zero-2: Infiltración de Crimen Organizado  
**Misión:** Narcos intentan usar la app para marcar territorio o desinformar.  
**Responsables:** Trust & Safety, OpSec, Justicia Restaurativa.

## 1. Detección de "Marcaje Territorial"
- **Patrón de Ataque:** Bandas criminales reportan "Luminaria mala" falsamente en esquinas específicas para marcar zonas de venta o ahuyentar patrullaje.
- **Defensa:** Anulación de reportes repetitivos de usuarios nuevos en zonas de "Alto Riesgo" sin validación fotográfica verificada por IA (meta-data análisis).

## 2. Desinformación Táctica (Cops & Robbers)
- **Ataque:** Reportar falsos accidentes para desviar a Carabineros.
- **Defensa:** Integración con Waze/SOSAFE para validación cruzada. Si Civicum es la ÚNICA fuente de un incidente crítico, se marca como "No Confirmado" y no se notifica a autoridades hasta tener 3 fuentes independientes.

## 3. Protección de Soplones (Whistleblowers)
- Si un usuario reporta crimen organizado, su reporte se anonimiza a nivel de base de datos inmediatamente (Hashing de ID) y se aplica un "Retraso de Publicación" aleatorio para evitar correlación temporal con la identidad del denunciante.

*Civicum es para vecinos, no para soldados.*
