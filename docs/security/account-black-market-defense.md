# Protocolo Mercado Negro (Squad Finanzas-3)

**Squad:** 173. Escuadrón Mercado Negro: Venta de Cuentas  
**Misión:** Usuarios vendiendo cuentas "verificadas" o con alto Karma en el mercado negro.  
**Responsables:** IAM, Security, Reputacional.

## 1. Detección de "Cambio de Manos"
- **Trigger:** Una cuenta activa en Arica (Norte) cambia repentinamente a Punta Arenas (Sur) y cambia radicalmente su comportamiento de escritura (NLP) y horarios.
- Acción: Se exige re-verificación biométrica o ClaveÚnica. Si falla, baneo permanente.

## 2. Scraping Defensivo
- Un bot de Civicum escanea sitios como Facebook Marketplace, Reddit y foros de hacking buscando "Vendo cuenta Civicum nivel Oro".
- Si encontramos una venta, compramos la cuenta (operación encubierta) para banear al vendedor y analizar la red.

## 3. Intransferibilidad de Reputación
- La reputación NO es transferible. Si cambias de RUT o email, empiezas de cero.
- No existe el concepto de "fusión de cuentas" para evitar acumulación de poder comprado.

*La confianza se gana, no se compra.*
