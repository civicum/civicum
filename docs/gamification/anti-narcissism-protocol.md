# Protocolo Anti-Narciso (Squad Psicología-1)

**Squad:** 151. Escuadrón Narciso: El EGO del Usuario  
**Misión:** ¿Estamos gamificando la participación al punto de crear "influencers cívicos" tóxicos?  
**Responsables:** Growth, Moderation Policy, Riesgos.

## 1. Cero "Ranking Global"
- **Prohibido:** Leaderboards de "Usuarios con más reportes". Esto incentiva el spam y reportes falsos para subir de nivel.
- **Permitido:** "Tu barrio ha logrado X". El éxito es colectivo.

## 2. Karma Invisible (Shadow-Metric)
- El "Nivel de Reputación" es una métrica interna para priorización de reportes, pero NO es visible públicamente en el perfil del usuario.
- Evita que usuarios compitan por "quién tiene el perfil más dorado".

## 3. Desactivación de "Likes" en Denuncias
- En reportes sensibles (Seguridad/Crimen), se desactiva el contador de "Apoyos" visual. Solo se muestra "Vecinos han validado esto".
- Evita la dopamina barata de ver subir el numerito en temas serios.

*El héroe es el barrio, no el usuario.*
