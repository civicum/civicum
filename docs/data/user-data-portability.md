# Protocolo de Exportabilidad Real (Squad Orwell-5)

**Squad:** 85. Escuadrón Orwell-5: Exportabilidad Real  
**Misión:** ¿Son útiles los Excel/PDF que exportamos?  
**Responsables:** Data Analyst, Service Designer, Open Data, Arquitecto Info.

## 1. La Prueba de la Tabla Dinámica
Un export no sirve si tienes que pasar 3 horas limpiándolo.
- **Estándar:** El CSV exportado debe poder abrirse en Excel y permitir hacer una Tabla Dinámica en < 1 minuto sin limpiar celdas fusionadas ni formatos extraños.

## 2. JSON Completo (Takeout)
- El usuario puede descargar un JSON con TODO su historial de interacciones, votos, reportes y metadatos.
- Estructura documentada (Schema.org compatible donde sea posible).

## 3. Interoperabilidad Municipal
- Los reportes se exportan en formatos estándar de la industria (Shapefile, GeoJSON) para que los departamentos de obras municipales puedan cargarlos directamente en sus sistemas GIS (ArcGIS, QGIS).
