# Protocolo de Propiedad Intelectual y Scraping (Squad Kafka-Epsilon)

**Squad:** 70. Escuadrón Kafka-Epsilon: Propiedad Intelectual  
**Misión:** ¿Estamos infringiendo copyright con los datos scrapeados?  
**Responsables:** Abogado PI, Licencias Open Data, Fuentes Públicas, Compliance.

## 1. Doctrina de "Hechos, no Obras"
Los datos públicos (nombres de calles, sueldos de funcionarios, textos de leyes) son hechos, no obras creativas protegidas por derecho de autor.
- **Postura Legal:** El Estado no tiene copyright sobre la realidad administrativa. Scrapeamos con confianza.

## 2. Respeto a los Términos de Uso (Robots.txt)
- Intentamos respetar `robots.txt` por cortesía, pero si bloquea el acceso a información pública obligatoria por Ley de Transparencia, lo ignoramos éticamente (Civil Disobedience coding).
- **Rate Limit:** Scrapeamos lento para no causar DDOS ("Be Nice").

## 3. Licenciamiento de Dataset Derivados
Si Civicum cura, limpia y enriquece una base de datos pública:
- **Licencia Salida:** CC-BY-SA 4.0.
- **Atribución:** Siempre citamos la fuente original ("Fuente: Datos brutos de Municipalidad de X, procesados por Civicum").

*Los datos pagados con impuestos son del dominio público.*
